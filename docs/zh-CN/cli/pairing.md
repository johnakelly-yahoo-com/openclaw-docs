# `openclaw pairing`

13. 批准或检查 DM 配对请求（适用于支持配对的渠道）。

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
