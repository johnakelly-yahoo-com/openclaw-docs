## 21. 完整列表

- 22. 通过 WhatsApp Web（Baileys）进行 WhatsApp 集成
- 23. Telegram 机器人支持（grammY）
- 24. Discord 机器人支持（channels.discord.js）
- 25. Mattermost 机器人支持（插件）
- 26. 通过本地 imsg CLI（macOS）集成 iMessage
- 27. 面向 Pi 的代理桥接，RPC 模式并支持工具流式传输
- 28. 对长响应的流式与分块处理
- 29. 针对每个工作区或发送者的隔离会话的多智能体路由
- 30. 通过 OAuth 进行 Anthropic 和 OpenAI 的订阅认证
- 31. 会话：直接聊天会折叠到共享的 `main`；群组彼此隔离
- 32. 支持群聊，并基于提及进行激活
- 33. 支持图像、音频和文档等媒体
- 34. 可选的语音备忘录转写钩子
- 35. WebChat 和 macOS 菜单栏应用
- 36. 支持配对和 Canvas 界面的 iOS 节点
- 37. 支持配对、Canvas、聊天和相机的 Android 节点

<Note>
38. 已移除旧版 Claude、Codex、Gemini 和 Opencode 路径。 39. Pi 是唯一的
</Note>
