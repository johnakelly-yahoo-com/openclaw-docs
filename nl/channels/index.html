    Chatkanalen¶

    OpenClaw kan met je communiceren via elke chat-app die je al gebruikt. Elk kanaal verbindt via de Gateway. Tekst wordt overal ondersteund; media en reacties verschillen per kanaal.

    Ondersteunde kanalen¶

    • WhatsApp — Meest populair; gebruikt Baileys en vereist QR-koppeling.
    • Telegram — Bot API via grammY; ondersteunt groepen.
    • Discord — Discord Bot API + Gateway; ondersteunt servers, kanalen en DM's.
    • Slack — Bolt SDK; werkruimte-apps.
    • Feishu — Feishu/Lark-bot via WebSocket (plugin, afzonderlijk geïnstalleerd).
    • Google Chat — Google Chat API-app via HTTP-webhook.
    • Mattermost — Bot API + WebSocket; kanalen, groepen, DM's (plugin, afzonderlijk geïnstalleerd).
    • Signal — signal-cli; privacygericht.
    • BlueBubbles — Aanbevolen voor iMessage; gebruikt de BlueBubbles macOS-server REST API met volledige functiesteun (bewerken, intrekken, effecten, reacties, groepsbeheer — bewerken momenteel defect op macOS 26 Tahoe).
    • iMessage (legacy) — Verouderde macOS-integratie via imsg CLI (verouderd; gebruik BlueBubbles voor nieuwe installaties).
    • Microsoft Teams — Bot Framework; ondersteuning voor enterprises (plugin, afzonderlijk geïnstalleerd).
    • LINE — LINE Messaging API-bot (plugin, afzonderlijk geïnstalleerd).
    • Nextcloud Talk — Zelfgehoste chat via Nextcloud Talk (plugin, afzonderlijk geïnstalleerd).
    • Matrix — Matrix-protocol (plugin, afzonderlijk geïnstalleerd).
    • Nostr — Gedecentraliseerde DM's via NIP-04 (plugin, afzonderlijk geïnstalleerd).
    • Tlon — Urbit-gebaseerde messenger (plugin, afzonderlijk geïnstalleerd).
    • Twitch — Twitch-chat via IRC-verbinding (plugin, afzonderlijk geïnstalleerd).
    • Zalo — Zalo Bot API; populaire messenger in Vietnam (plugin, afzonderlijk geïnstalleerd).
    • Zalo Personal — Persoonlijk Zalo-account via QR-login (plugin, afzonderlijk geïnstalleerd).
    • WebChat — Gateway WebChat-UI via WebSocket.

    Notities¶

    • Kanalen kunnen gelijktijdig draaien; configureer er meerdere en OpenClaw routeert per chat.
    • De snelste installatie is meestal Telegram (eenvoudige bot-token). WhatsApp vereist QR-koppeling en slaat meer status op schijf op.
    • Groepsgedrag verschilt per kanaal; zie Groepen.
    • DM-koppeling en toegestane lijsten worden afgedwongen voor veiligheid; zie Beveiliging.
    • Telegram-internals: grammY-notities.
    • Problemen oplossen: Problemen met kanalen.
    • Modelproviders worden afzonderlijk gedocumenteerd; zie Modelproviders.
