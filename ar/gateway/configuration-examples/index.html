    أمثلة التهيئة¶

    تتوافق الأمثلة أدناه مع مخطط التهيئة الحالي. وللاطلاع على المرجع الشامل وملاحظات كل حقل، راجع التهيئة.

    البدء السريع¶

    الحد الأدنى المطلق¶

    {
  agent: { workspace: "~/.openclaw/workspace" },
  channels: { whatsapp: { allowFrom: ["+15555550123"] } },
}

    احفظ إلى ~/.openclaw/openclaw.json ويمكنك مراسلة البوت عبر الرسائل الخاصة من ذلك الرقم.

    بداية موصى بها¶

    {
  identity: {
    name: "Clawd",
    theme: "helpful assistant",
    emoji: "🦞",
  },
  agent: {
    workspace: "~/.openclaw/workspace",
    model: { primary: "anthropic/claude-sonnet-4-5" },
  },
  channels: {
    whatsapp: {
      allowFrom: ["+15555550123"],
      groups: { "*": { requireMention: true } },
    },
  },
}

    وضع الرسائل الخاصة الآمن (صندوق وارد مشترك / رسائل خاصة متعددة المستخدمين)¶

    إذا كان بإمكان أكثر من شخص مراسلة البوت عبر الرسائل الخاصة (وجود عدة إدخالات في allowFrom، أو موافقات إقران لعدة أشخاص، أو dmPolicy: "open")، فعِّل وضع الرسائل الخاصة الآمن حتى لا تشترك الرسائل الخاصة من مرسلين مختلفين في سياق واحد افتراضيًا:

    {
  // Secure DM mode (recommended for multi-user or sensitive DM agents)
  session: { dmScope: "per-channel-peer" },

  channels: {
    // Example: WhatsApp multi-user inbox
    whatsapp: {
      dmPolicy: "allowlist",
      allowFrom: ["+15555550123", "+15555550124"],
    },

    // Example: Discord multi-user inbox
    discord: {
      enabled: true,
      token: "YOUR_DISCORD_BOT_TOKEN",
      dm: { enabled: true, allowFrom: ["alice", "bob"] },
    },
  },
}

    نصائح¶

    • إذا قمت بتعيين dmPolicy: "open"، فيجب أن تتضمن قائمة allowFrom المطابقة "*".
    • تختلف معرّفات الموفّر (أرقام الهواتف، معرّفات المستخدمين، معرّفات القنوات). استخدم وثائق الموفّر لتأكيد الصيغة.
    • أقسام اختيارية لإضافتها لاحقًا: web، browser، ui، discovery، canvasHost، talk، signal، imessage.
    • راجع الموفّرون واستكشاف الأخطاء وإصلاحها لملاحظات إعداد أعمق.
